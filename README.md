# Khodar Starbucks Collection

Public website with an interactive world map of my Starbucks mug collection.

## Project idea

This project will show:
- countries where my mugs came from;
- which mugs belong to each country;
- who brought them;
- when they were added;
- photos of the mugs.

## Planned stack

- Frontend: React + Vite
- Map: React Simple Maps
- Data: JSON
- Hosting: Vercel
- Code repository: GitHub
